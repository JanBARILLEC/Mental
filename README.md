# Mental Calculus

A free, browser-based app to train mental math — no installation, no account, works on mobile.

**Live demo:** `https://your-username.github.io/calcul-mental`

---

## Features

- **Timed practice** — Addition, Subtraction, Multiplication, Division, or Mixed
- **5 difficulty levels** — from single-digit to 6-digit numbers
- **Per-question timer** — 10s / 20s / 30s or disabled
- **10 technique lessons** — complement method, cross multiplication, squaring trick, percentage tricks and more
- **Progress tracking** — accuracy, speed, streaks, charts (stored locally in the browser)
- **3 languages** — English, French, German

---

## How to use locally

No build step required. Just open `index.html` in your browser.

```
calulus/
├── index.html   # Structure
├── style.css    # Dark theme
└── app.js       # All logic + translations + lessons
```

---

## Deploy to GitHub Pages (free hosting)

1. Create a public repository on [github.com](https://github.com)
2. Upload the 3 files (`index.html`, `style.css`, `app.js`)
3. Go to **Settings → Pages → Branch: main → Save**
4. Your site is live at `https://your-username.github.io/your-repo-name`

---

## Tech stack

Plain HTML / CSS / JavaScript — no framework, no dependencies.
